# WB Promotion Analytics - Сервис визуализации данных

## 📋 Описание
Сервис для работы с данными рекламных кампаний Wildberries (раздел WB продвижение). Позволяет вводить данные вручную и загружать расширенную статистику из Excel файлов.

## ✨ Возможности

### 1. Ручной ввод данных кампаний
- **Название кампании** - автоматически извлекается из URL ссылки
- **Тип кампании** - CPC (оплата за клики) или CPM (оплата за 1000 показов)
- **Затраты** - в рублях
- **Показы** - количество
- **CTR** - в процентах
- **Заказы** - в штуках
- **Доля затрат** - в процентах
- **Период** - с обязательным указанием недели (ПН-ВС)

### 2. Загрузка Excel файлов
- Drag & Drop зона для загрузки файлов
- Привязка периода (ДД.ММ.ГГГГ - ДД.ММ.ГГГГ) к каждому файлу
- Просмотр загруженных файлов
- Отображение расширенной статистики из файлов

### 3. Фильтрация и аналитика
- Фильтр по датам с ограничением на неделю (ПН-ВС)
- Быстрая навигация по неделям (предыдущая/текущая/следующая)
- Сводные карточки статистики:
  - Общие затраты
  - Количество показов
  - Средний CTR
  - Количество заказов

### 4. Хранение данных
- Все данные сохраняются в localStorage браузера
- Сохранение persists между сессиями

## 🚀 Использование

1. Откройте `index.html` в любом современном браузере
2. Нажмите "+ Добавить кампанию"
3. Заполните форму данными из WB Promotion
4. Для загрузки расширенных данных нажмите "📊 Статистика" на карточке кампании
5. Перетащите Excel файл или выберите через диалог
6. Укажите период для файла (по умолчанию берётся период кампании)

## 📝 Формат данных

### Кампания
```json
{
  "id": 1234567890,
  "name": "Название кампании",
  "url": "https://...",
  "type": "CPC",
  "cost": 1000.50,
  "impressions": 50000,
  "ctr": 2.5,
  "orders": 15,
  "share": 10.5,
  "period": {
    "start": "2026-03-16",
    "end": "2026-03-22"
  },
  "files": [],
  "createdAt": "2026-03-20T10:00:00.000Z"
}
```

### Загруженный файл
```json
{
  "id": 1234567891,
  "name": "report.xlsx",
  "period": {
    "start": "2026-03-16",
    "end": "2026-03-22"
  },
  "data": [...],
  "uploadedAt": "2026-03-20T10:00:00.000Z"
}
```

## ⚠️ Ограничения

- Данные хранятся только в браузере пользователя
- При очистке localStorage данные будут потеряны
- Максимальный размер файла зависит от браузера
- Таблица расширенной статистики показывает первые 100 записей
